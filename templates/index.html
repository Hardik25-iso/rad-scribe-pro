How It Works
Model D Pipeline

From raw chest X-ray to a clinically grounded, confidence-scored report in four stages.

1
Dual Visual Encoding
ViT-Base/16 captures global spatial structure. DenseNet-121 captures local pathology texture. Both fused into 1024-dim via LayerNorm projection.
ViT + DenseNet → 1024-dim
2
Counterfactual Retrieval
FAISS searches 4,300+ training embeddings. Returns top-3 similar cases AND top-2 dissimilar cases for contrast.
FAISS IndexFlatIP
3
BioGPT Generation
Image token prepended to retrieved context. BioGPT (15M PubMed papers) generates a grounded report with beam search.
microsoft/biogpt
4
Confidence Scoring
Each sentence gets a log-probability score. Green = high confidence. Red = flagged for radiologist verification.
Sentence-level LP
Research Contributions
Novel on IU X-Ray Dataset

Five firsts, each independently justified by ablation study results.

01
BioGPT Decoder
First application of Microsoft BioGPT on IU X-Ray. Medical terms like cardiomegaly and atelectasis are single tokens — not fragmented subwords. Directly improves generation quality.
FIRST ON IU X-RAY
02
Counterfactual RAG
First counterfactual retrieval on IU X-Ray. Retrieves similar AND dissimilar cases — forcing the model to learn what distinguishes each scan from normal or abnormal presentations.
FIRST ON IU X-RAY
03
Dual Encoder Fusion
ViT-Base/16 for global attention + DenseNet-121 for local pathology detection, concatenated and projected to 1024-dim. Global and local understanding simultaneously.
FIRST ON IU X-RAY
04
Claim-Level Verification
Model E scores each generated sentence against three evidence sources: classifier probability, supportive case evidence, and counterfactual case evidence.
NOVEL CONTRIBUTION
05
Sentence Confidence Display
Colour-coded confidence per sentence in the clinical dashboard. Radiologists see which findings are high-confidence and which require manual verification — reducing cognitive load.
CLINICAL INNOVATION
06
CheXBert Evaluation
Model E evaluated on all 14 clinical conditions using StanfordAIMI CheXBert. 78.25 MicroF1 — directly measuring diagnostic accuracy across specific pathologies.
